MusicTab Guide helps to search and download tabs from Internet using custom web search engine. This feature is experimental due to instability of external web service. To use this feature you need to choose your group and press Search tabs button. You will be redirected to Web search page, then search will be launched automatically. The results will be shown after search complete. Select desired tabs and click Download button. After this tabs should be downloaded and add to the group. Use Previous and Next buttons to navigate between different result pages it they are available.
