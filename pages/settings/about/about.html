MusicTab Guide

Version 3.0 released in January, 2013

This application allows you to search, organize and view tabs stored in the following formats:

  • Guitar Pro (gp3, gp4, gp5, gpx)
  • MusicXml (xml)

Also it allows you to create your custom tabs using vextab language and provides some additional tools to improve your guitar skills.

Rate MusicTab Guide if you are interesting in next versions with new features and improvements.

This application uses vexflow library to render stave notes and chords.
